# Locale files

OrchardBridge loads UI strings from JSON files in this folder.
Keys are Traditional Chinese source strings. Values are localized UI strings.
If a key is missing, the app falls back to English so the interface remains usable.

目前語言檔以繁體中文原文字串作為 key，value 為各語言翻譯。若某個 key 缺漏，程式會自動使用英文備援。
